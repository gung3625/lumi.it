    STEP 2 / 5 · 핵심 단계

    어느 마켓에 올리시겠어요?

    사장님이 이미 쓰는 마켓을 연결합니다. 처음이면 1개만 연결해도 시작할 수 있어요.

    쿠팡 윙

    Vendor ID + Access Key + Secret Key

    쿠팡 OPEN API 키 발급 안내
    1. 쿠팡 Wing 로그인 → 우상단 [판매자명]
    2. [추가판매정보] → [OPEN API 키 발급]
    3. 약관 동의 → 사용 목적: OPEN API 선택
    4. 발급된 Vendor ID / Access Key / Secret Key를 아래에 입력

    서버에서 암호화 저장합니다. 사장님 화면에는 마지막 4자리만 표시돼요.

    네이버 스마트스토어

    Application ID + Secret

    네이버 커머스 API 등록 안내
    1. 네이버 커머스 API 센터 로그인
    2. [애플리케이션 등록] → 사용자 직접 사용 (SELF) 선택
    3. 발급된 Application ID / Secret을 아래에 입력
    4. 스코프(상품/주문 등)를 활성화해야 판매 권한이 생겨요

    연결 후 판매 권한 확인은 백그라운드로 진행됩니다. 권한이 부족해도 다음 단계로 넘어갈 수 있고, 결과가 도착하면 알림으로 알려드릴게요.
