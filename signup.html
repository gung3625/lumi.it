    사장님이 이미 쓰는 마켓을 연결합니다. 처음이면 1개만 연결해도 시작할 수 있어요.

    쿠팡 윙

    Vendor ID + Access Key + Secret Key

    1 2 3 4 5

    예상 소요 30초

    쿠팡 키 발급하러 가요

    쿠팡 Wing에서 OPEN API 키를 발급받고, 두 키를 복사해 주세요.

    1. 쿠팡 Wing 로그인 → 우상단 [판매자명]
    2. [추가판매정보] → [OPEN API 키 발급]
    3. 약관 동의 → 사용 목적: OPEN API 선택
    4. 발급된 Vendor ID / Access Key / Secret Key를 복사
    쿠팡 페이지에서 두 키를 복사해 주세요
    1. 발급된 Vendor ID 복사
    2. Access Key 복사
    3. Secret Key 복사 (가장 긴 키)

    루미로 돌아오시면 자동으로 감지해 입력해 드릴게요.

    서버에서 암호화 저장합니다. 사장님 화면에는 마지막 4자리만 표시돼요.

    네이버 스마트스토어

    Application ID + Secret

    1 2 3 4 5

    예상 소요 60초

    네이버 키 발급하러 가요

    네이버 커머스 API 센터에서 애플리케이션을 등록하고, ID와 Secret을 복사해 주세요.

    1. 네이버 커머스 API 센터 로그인
    2. [애플리케이션 등록] → 사용자 직접 사용 (SELF) 선택
    3. 발급된 Application ID / Secret을 복사
    4. 스코프(상품/주문 등)를 활성화해야 판매 권한이 생겨요
    네이버 페이지에서 두 키를 복사해 주세요
    1. Application ID 복사
    2. Application Secret 복사 (긴 키)

    루미로 돌아오시면 자동으로 감지해 입력해 드릴게요.
