인스타 비즈니스 계정이 꼭 필요한가요?

네, 인스타 정책상 외부 도구가 게시·인사이트를 가져오려면 비즈니스/크리에이터 계정이어야 합니다. 개인 계정에서 비즈니스로 전환은 무료, 3-5분이면 끝나요.

가격이 얼마예요? 언제부터 돈을 내나요?

지금은 모든 기능 무료입니다. 카카오 로그인 한 번이면 결제 정보 입력 없이 바로 시작할 수 있어요.

유료 플랜 도입 시 단계는 정확히 다음과 같습니다:

  1. 가격·플랜 내용 사장님께 이메일 + 인앱 공지 (최소 30일 전)
  2. 사장님이 직접 "유료 플랜 시작" 동의 (체크박스 + 결제 정보 입력)
  3. 동의한 사장님만 결제 진행. 동의 X = 무료 플랜 또는 서비스 사용 중단

사장님 모르게 결제되는 일은 없습니다. 자세한 약관은 이용약관 제14조 · 환불정책 참고.

자동 게시는 어느 정도 자동인가요?

3가지 모드:

  • 즉시 게시 — 사진 + 캡션 확인 → 바로 게시
  • 예약 게시 — 사장님이 지정한 시간에 자동
  • 베스트 시간 자동 — 사장님 팔로워가 가장 활발한 시간 분석 후 자동

모든 게시는 사장님이 사전에 확인 가능. 사진 + 캡션 결정권은 사장님 100%.

루미가 캡션 쓰는 건 ChatGPT 와 뭐가 다른가요?

3가지가 다릅니다:

  1. 매장 톤 학습 — 사장님이 좋아한 캡션을 기억해서 다음 캡션이 점점 사장님 톤에 맞춰져요. ChatGPT 는 매 대화마다 매장 설명을 다시 입력해야 해요.
  2. 한국 트렌드 해시태그 — 네이버 실시간 검색량 기반. ChatGPT 는 옛 데이터.
  3. 자동 게시 + 베스트 시간 — 캡션 만드는 데서 끝이 아니라 게시까지 자동.
내 사진·캡션 데이터는 안전한가요?

네. 모든 데이터는 Supabase (한국 서울 리전 우선) 에 암호화 저장. 인스타 액세스 토큰은 Vault 에 AES-256 으로 별도 보관. OpenAI API 는 기본 정책상 lumi 데이터를 모델 학습에 사용하지 않아요 (Default Off).

사장님이 직접 검수 / 수정할 수 있나요?

네. lumi 가 캡션 3개를 추천하면 사장님이 그중 하나 고르거나 직접 수정한 뒤 게시. 자동 모드 (즉시 / 베스트 시간) 도 게시 전 사장님이 미리 캡션 확인할 수 있어요.

매번 검수 부담 줄이려면 👍·👎 평가로 매장 톤 학습 → 시간 지나면 사장님이 손 댈 일 점점 줄어듭니다.

탈퇴는 어떻게 하나요?

설정 → 회원 탈퇴 버튼 한 번이면 끝. 7일 유예 기간 동안 같은 버튼으로 즉시 복구 가능. 7일 후 자동 영구 삭제. 즉시 삭제 원하시면 lumi@lumi.it.kr 으로 요청.

한 달에 인스타 몇 건 올릴 수 있나요?

매장 운영 패턴에 맞춰 사장님이 결정. 매일 1-3건 권장 (인스타 알고리즘이 너무 자주 올리는 계정 노출 ↓). 예약 게시 + 베스트 시간 자동 발행으로 사장님 손 안 거치고 꾸준한 게시 가능.
