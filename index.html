FAQ
무료 체험 끝나면 자동 결제?+
아니요. 자동 결제 없이 직접 선택하셔야 전환돼요. 카드 등록도 필요 없어요.
AI 쓴 티 안 나나요?+
업종·분위기·말투를 학습해서 써요. 피드백 줄수록 자연스러워져요.
내 사진 다른 곳에 쓰이나요?+
절대 아닙니다. 캡션 생성에만 사용되고, 외부 학습에 사용되지 않아요.
비밀번호 넘겨야 하나요?+
Meta 공식 API 사용, 비밀번호 저장 안 해요.
하루에 몇 개까지?+
캡션 생성은 무제한. 자동 게시는 하루 최대 5개, 예약도 가능해요.
ChatGPT로 직접 쓰면?+
lumi는 사진 분석 + 트렌드 태그 + 바로 게시. ChatGPT는 복붙만 15분 이상.
대행사보다 나은 점?+
대행사 월 50~100만원, lumi 월 1.9만원부터. 수정도 바로 가능.
인스타 말고 다른 SNS?+
현재 인스타 전용. 네이버 블로그·플레이스 곧 지원 예정.
