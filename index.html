글 고민 0분

매일 30분씩 글 쓰던 시간, 이제 사진 한 장이면 끝. 캡션이 바로 나오니까 촬영에만 집중하세요.

노출 높이는 태그

요즘 사람들이 검색하는 태그를 골라 달아줘요. 같은 사진이어도 태그 하나로 노출이 2~3배 차이나요.

오늘 날씨가 곧 콘텐츠

비 오는 날 빵 사진 올리면 "이런 날엔 따뜻한 빵이죠" 같은 글이 나와요. 날씨가 공감 포인트가 돼요.

최적 시간에 자동 게시

밤 11시에 올릴 필요 없어요. 우리 팔로워가 가장 많이 보는 시간에 자동으로 올라가요.

우리 매장 말투로만

감성 카페는 감성으로, 힙한 헤어샵은 힙하게. 피드백 줄수록 사장님 말투에 가까워져요.

여러 장도 한 번에

사진 10장까지 한 번에 올리면 캐러셀 게시물로 완성. 하나하나 따로 올릴 필요 없어요.

스토리 자동 게시

피드에 올리면 스토리에도 자동 공유. 팔로워에게 하루 두 번 노출.

주변 행사·축제 반영

근처 벚꽃 축제, 음식 페스티벌이 있으면 캡션에 자연스럽게 녹여서 써드려요.

lumi 링크 페이지

매장 위치, 전화, 주문, 예약을 한 페이지에. 무료로 제공.

7일 콘텐츠 캘린더

"오늘 뭐 찍지?" 고민 끝. 업종과 지역 기반으로 일주일 촬영 계획을 짜드려요.

쓸수록 배우는 말투

피드백 한 번이면 다음엔 더 사장님 스타일로. 예시 캡션을 등록하면 그 톤을 학습해요.

자주 묻는 질문

무료 체험 끝나면 자동 결제되나요?+
아니요. 베타 기간 종료 최소 2주 전에 안내드리고, 자동 결제 없이 직접 선택하셔야 전환돼요. 카드 등록도 필요 없어요.
AI가 쓴 티 안 나나요?+
lumi는 우리 매장 업종·분위기·말투를 학습해서 캡션을 써요. 생성 후 직접 수정도 가능하고, 피드백을 줄수록 더 자연스러워져요.
내 사진이 다른 곳에 쓰이진 않나요?+
절대 아닙니다. 업로드하신 사진은 캡션 생성에만 사용되고, 외부 학습에 활용되지 않아요. 언제든 삭제 요청도 가능해요.
비밀번호 넘겨야 하나요?+
Meta 공식 API를 사용하고, 비밀번호를 저장하지 않아요. 연동 해제도 설정에서 원클릭으로 가능해요.
하루에 몇 개까지 올릴 수 있나요?+
캡션 생성은 무제한이에요. 인스타 자동 게시는 하루 최대 5개까지 가능하고, 예약 게시도 지원해요.
ChatGPT로 직접 쓰면 안 되나요?+
lumi는 사진을 자동 분석하고, 트렌드 해시태그를 달고, 인스타에 바로 게시까지 해줘요. ChatGPT는 복사-붙여넣기만 15분 이상 걸려요.
대행사보다 나은 점이 뭔가요?+
대행사는 월 50~100만원, lumi는 월 1.9만원부터. 수정도 직접 바로 가능하고, 우리 매장 말투로 매일 올릴 수 있어요.
인스타 말고 다른 SNS도 되나요?+
현재는 인스타그램 전용이에요. 네이버 블로그, 네이버 플레이스는 곧 지원 예정이에요.
