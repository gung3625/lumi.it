각 기능이 뭔지 알아보세요

루미의 모든 기능을 하나하나 설명해드려요.

캡션 생성

월 캡션 횟수

한 달 동안 생성할 수 있는 캡션의 총 횟수예요. 사진이나 영상 한 건당 캡션 1개가 차감돼요. 스탠다드 12회, 프로 20회, 비즈니스 30회로 월 게시 빈도에 맞게 선택하세요. 횟수를 초과하면 다음 달 갱신 때 자동으로 충전돼요.

스탠다드프로비즈니스
캡션 재생성

캡션 재생성 (건당 최대 3회)

처음 받은 캡션이 마음에 안 들면 같은 사진으로 다시 생성할 수 있어요. 건당 최대 3번까지 재생성 가능하고, 재생성은 캡션 횟수에서 추가로 차감되지 않아요. 코멘트 칸에 원하는 방향을 적으면 다음 생성부터 반영해줘요.

스탠다드프로비즈니스
게시 자동화

자동 게시

캡션을 확인하고 "게시" 버튼을 누르면 루미가 인스타그램에 바로 올려줘요. 직접 인스타 앱을 열 필요 없이, 루미 화면에서 모든 게 끝나요. Meta 공식 API를 통해 연결되기 때문에 계정이 안전하게 유지돼요.

스탠다드프로비즈니스
릴스

릴스 자동 생성

사진 여러 장을 올리면 루미가 자동으로 릴스(짧은 영상)로 만들어줘요. 영상 편집 지식 없이도 트렌디한 릴스를 손쉽게 게시할 수 있어요. 배경 음악 선택과 장면 전환 효과도 자동으로 적용돼요.

스탠다드프로비즈니스
루미 학습

말투 자동 학습

게시할 때마다 루미가 사장님의 말투를 조금씩 배워나가요. 이전에 올린 캡션을 평가해주면, 다음 번 캡션부터 더 우리 가게답게 써줘요. 20개 게시물이 쌓이면 완전히 우리 매장 고유의 말투가 형성돼요.

스탠다드프로비즈니스
트렌드

트렌드 키워드

네이버, 구글, 유튜브, 인스타그램, 핀터레스트에서 실시간으로 수집한 트렌드를 캡션에 자동 반영해줘요. 업종별로 분류된 키워드만 골라서 활용하기 때문에 "요즘 뜨는 말"이 자연스럽게 녹아들어요.

스탠다드프로비즈니스
해시태그

해시태그 자동 생성

업종과 게시물 내용에 딱 맞는 해시태그를 자동으로 붙여줘요. 너무 많이 붙이거나 관련 없는 태그를 다는 실수를 막아주고, 조회수에 도움이 되는 태그를 조합해서 제안해요.

스탠다드프로비즈니스
예약 게시

예약 게시

원하는 날짜와 시간을 지정해두면 루미가 그 시간에 알아서 올려줘요. 여러 게시물을 미리 만들어 둘 수 있어 한꺼번에 1주일치 콘텐츠를 준비하는 것도 가능해요. 바쁜 시간대에 게시 알림을 따로 신경 쓸 필요가 없어요.

프로비즈니스
캘린더

캘린더

예약된 게시물을 달력 형태로 한눈에 보여줘요. 이번 주 어떤 콘텐츠가 올라가는지, 빈 날은 없는지 바로 확인할 수 있어요. 드래그 앤 드롭으로 일정을 조정하는 것도 지원할 예정이에요.

프로비즈니스
최적 시간

최적 시간 자동 게시

팔로워들이 가장 많이 활동하는 시간을 매일 분석해서 알려줘요. 예약할 때 "최적 시간"을 선택하면 루미가 자동으로 그 시간에 게시해요. 직접 데이터를 분석할 필요 없이, 노출이 잘 되는 타이밍을 놓치지 않을 수 있어요.

프로비즈니스
링크인바이오

링크인바이오

인스타그램 프로필 링크 하나로 여러 주소를 안내할 수 있는 페이지를 만들어줘요. 메뉴판, 예약 링크, 쇼핑몰, 카카오채널 등을 한 화면에 묶어서 팔로워들이 헷갈리지 않게 해줘요. 루미에서 직접 편집하고 바로 적용돼요.

프로비즈니스
DM 자동화

DM 자동응답

고객이 보낸 DM에 루미가 자동으로 답장해줘요. 프로 플랜은 특정 키워드(예: "예약", "가격")가 포함된 메시지에 미리 설정한 답변을 보내는 키워드 매칭 방식이에요. 비즈니스 플랜은 루미가 대화 맥락을 이해하고, 의도와 감정을 분류해 자연스러운 답변을 자동 생성해요. 매장 정보를 학습한 루미가 운영시간·위치·메뉴 문의까지 알아서 처리해줘요.

프로비즈니스
댓글 자동화

댓글 자동응답

게시물에 달린 댓글에 루미가 자동으로 답글을 달아줘요. 스팸이나 악성 댓글은 걸러내고, 진짜 문의나 반응에만 친절하게 답해요. 사장님 말투를 학습한 루미가 응답하기 때문에 기계 같지 않고 자연스러운 대화처럼 보여요.

비즈니스
루미 분류

루미 의도 + 감정 분류

DM이나 댓글을 받으면 루미가 먼저 "이 사람이 뭘 원하는지(의도)"와 "어떤 감정 상태인지"를 분류해요. 예약 문의인지, 불만인지, 단순 칭찬인지를 구분해서 상황에 맞는 답변을 생성해요. 사장님은 대시보드 로그에서 분류 결과를 확인할 수 있어요.

비즈니스
매장 컨텍스트

매장 컨텍스트 루미 답변

매장 정보(운영시간, 메뉴, 위치, 예약 방법 등)를 한 번 입력해두면 루미가 이를 바탕으로 답변을 생성해요. "몇 시까지 해요?", "주차 있어요?" 같은 반복 질문에 사람 대신 루미가 정확하게 답해줘요. 정보가 바뀌면 업데이트만 하면 바로 반영돼요.

비즈니스
스팸 차단

스팸 자동 차단

광고성 DM이나 악성 댓글을 루미가 자동으로 감지하고 차단해요. 팔로워를 낚으려는 이상한 계정이나 무의미한 스팸 메시지가 사장님께 도달하지 않도록 필터링해줘요. 차단 기준을 직접 설정하거나 루미 자동 판단에 맡길 수 있어요.

비즈니스
대시보드

사장님 대시보드 로그

루미가 처리한 DM·댓글 내역을 대시보드에서 한눈에 확인할 수 있어요. 어떤 답변을 보냈는지, 스팸으로 차단된 건 무엇인지, 의도 분류 결과는 어떤지 모두 기록돼요. 필요하면 특정 대화에 직접 개입해서 루미 답변을 수정할 수도 있어요.
