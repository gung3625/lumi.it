# 소싱 매입 프레임워크 (바이어 룰북)

> lumi 소싱봇이 "이 상품 매입할까?"를 판단하는 기준. 한국 이커머스 실수치 기반 (2026).
> 출처: 쿠팡 마켓플레이스 수수료 안내, dashpanda/ssjum/실무팩 셀러가이드, 네이버 스마트스토어 고객센터.

## Part 1 — 마진 계산법

### 1-1. 용어 (셀러가 제일 많이 헷갈리는 것)
- **마진** = 판매가 − 매입가 (단순 차액 — 이걸 수익으로 착각하면 망함)
- **순마진(순이익)** = 판매가 − 매입가 − **모든 판관비** ← 진짜 내 손에 남는 돈
- **마진율** = 순이익 ÷ 판매가 × 100 (★판매가 기준으로 고정해서 씀)
- **마크업** = 이익 ÷ 매입가 (도매상이 "2배 남는다" 할 때의 그것 — 마진율과 다름)

### 1-2. 숨은 비용 전부 (실수치)
| 비용 | 쿠팡 | 스마트스토어 |
|---|---|---|
| 판매수수료(카테고리) | 4~10.9% (컴퓨터 5 / 생활·가전 7.8 / 뷰티 9.6 / 패션 10.5 / 식품·가구 10.6~10.8) | 없음 |
| 결제/Npay 수수료 | 결제수수료 ~2.9%(2026, 윙 확인) | Npay 1.95~3.63% (매출등급별) |
| 채널/매출연동 | — | 네이버쇼핑 검색유입 시 ~2% |
| 부가세 VAT | 매출의 ~9.09% (매입세액 공제 가능) | 동일 |
| 택배비 | 2,600~3,500원/건 (편의점택배로 절감) | 동일 |
| 포장비 | 200~800원/건 | 동일 |
| 광고비 | 매출의 5~15% (CPC) | 동일 |
| 반품/불량 예비 | 카테고리별 2~20% | 동일 |
| 월 서비스료 | 55,000원 (월매출 100만+, 가전 500만+) | 없음 |

→ **실효 차감: 쿠팡 ≈ 판매가의 13~18%(수수료+결제+VAT), 스마트스토어 ≈ 5~8%.** 스토어가 수수료는 훨씬 쌈.

### 1-3. 공식
```
순이익 = 판매가 − 매입가 − 판매가×수수료율 − 택배 − 포장 − 판매가×광고율 − 판매가×반품율 − VAT부담
마진율 = 순이익 ÷ 판매가 × 100
판매가 역산 = 매입가 ÷ (1 − 목표마진율)   (1차 가격 잡고 순마진 재검증)
```

### 1-4. ⚠️ 핵심 함정 예시 (싸고 작은데 "3배 마진"인 상품)
화장품정리함: 매입 2,200 → 쿠팡 6,900 (3.1배, 좋아 보임). 단품·무료배송이면:
- 매입 2,200 + 쿠팡차감 18%(1,242) + 택배 3,000 + 포장 500 + 광고 10%(690) + 반품예비 ≈ **총 7,900 > 판매가 6,900 → 적자!**
- **택배비 3,000원이 저가품 마진을 통째로 잡아먹음.**
- → **해법: 3개 묶음 17,900원**(택배 1건으로 희석) 또는 유료배송. 그럼 순이익 ~3,000~4,000원 = OK.
- 교훈: **"마크업 배수"만 보면 안 됨. 택배비·수수료까지 넣은 순마진 + 객단가가 진짜다.**

### 1-5. 마진율 판단 기준
| 순마진율 | 판단 | 행동 |
|---|---|---|
| 40%+ | 안전 | 광고·재투자 여력 |
| 30~40% | 적정 | 반품·변동 여유 |
| 20~30% | 보통 | 객단가↑·박리다매 |
| 10~20% | 주의 | 반품 1건에 적자 가능 |
| 10%↓ | 위험 | **매입 제외** |

## Part 2 — 매입 결정 룰 (바이어로서)

### A. 게이트 — 하나라도 실패하면 즉시 제외
1. **순마진율 ≥ 25%** 또는 **건당 순이익 ≥ 3,000원** (둘 중 하나는 충족)
2. **규제/인증 통과**: 화장품(제조판매업·전성분), 식품(영업신고·유통기한 70%+), 전기전자(KC·전안법), 유아아동(KC), 의료기기(허가) — 미보유면 **제외**(법적 리스크)
3. **정품/상표 안전**: 브랜드 짝퉁·무단 병행수입 아님
4. **소싱 안정성**: 도매 재고 안정 + 최소구매수량(MOQ) 감당 + 리드타임 OK

### B. 스코어링 — 게이트 통과 후 점수화
**경쟁력 점수 = 수요 × 경쟁틈 × 마진 × 물류적합 × 차별화여지**
- **수요**: 네이버 검색량(실구매의도) + 쿠팡 판매·리뷰. ※검색량≠구매의도 (계산기·정보성 키워드 거름)
- **경쟁틈**: 1위 리뷰수 낮을수록↑(신규 진입 쉬움). 쿠팡 PB(코멧·탐사) 장악 시↓ — 단 품질/가격 우위 있으면 "고난도 도전"으로 살림
- **마진**: Part 1 순마진율
- **물류적합**: 작고 가벼움↑(택배·파손 유리), 부피·중량·파손위험↓
- **차별화여지**: 디자인·구성·번들 가능↑, 완전 commodity↓

### C. 제외 룰 (사지 말 것)
- 쿠팡 PB 초저가 장악 + 차별화 불가 (키친타월·고무장갑)
- 도매가 ≈ 쿠팡가 (마진 안 남음 — 노즈워크 1.2배)
- 부피 큰 저가품 (배송비가 마진 잠식 — 서랍장·리빙박스)
- 강한 브랜드 독점 + 내 우위 없음 (코코도르 디퓨저) ※우위 명확하면 고난도 도전
- 규제·인증 필요한데 미보유
- 시즌 끝물 (재고 리스크)

### D. 검증 → 사입 (리스크 관리)
1. 위탁/소량(5~10개)로 시작 → 2~4주 반응 확인
2. 잘 팔리는 것만 사입 확대 (20~50개)
3. 현금결제로 5~10% 추가할인 + 단골 형성
