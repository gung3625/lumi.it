# lumi 브랜드 보이스 가이드

## 핵심 속성 3가지

| 속성 | 설명 | O (이렇게) | X (이렇게 아님) |
|------|------|-----------|---------------|
| **친근함** | 사장님 옆에서 말하는 친구 톤 | "사진만 올리면 끝이에요" | "AI 기반 자동화 솔루션을 제공합니다" |
| **실용적** | 뜬구름 없이 숫자와 구체적 사례 | "37분 → 1분" | "효율적인 마케팅이 가능합니다" |
| **가벼움** | 부담 없고 쉬운 느낌 | "1분 체험해보기" | "지금 바로 가입하여 혜택을 누리세요" |

## 톤 매트릭스 (채널별)

| 채널 | 톤 | 예시 |
|------|-----|------|
| 랜딩 페이지 | 친근 + 확신 | "사진 한 장이면, 인스타는 끝이에요." |
| 이메일 | 개인적 + 따뜻 | "○○ 사장님, 첫 캡션을 만들어보세요" |
| 에러/실패 | 미안 + 해결책 | "잠시 문제가 생겼어요. 다시 시도해주세요" |
| CTA 버튼 | 행동 + 가벼움 | "무료로 시작하기", "첫 사진 올리러 가기" |
| FAQ | 대화체 + 안심 | "아니요, 비밀번호는 절대 필요하지 않아요" |
| 가격 페이지 | 구체적 + 비교 | "대행사 월 50만원 vs lumi 월 1.9만원" |

## 금지 표현

| 금지 | 이유 | 대안 |
|------|------|------|
| "혁신적인", "획기적인" | AI 스타트업 클리셰 | 구체적 수치로 대체 |
| "활용하세요" | AI가 쓴 티 남 | "써보세요", "해보세요" |
| "극대화", "최적화" | 차가운 기업 톤 | "더 잘 나와요", "딱 맞게" |
| "손쉽게", "간편하게" | 뻔함 | "1분이면", "사진만 올리면" |
| "강력한 기능" | 과장 | 기능을 직접 보여주기 |
| "~해 드리겠습니다" | 콜센터 톤 | "~해드려요" |

## 주어 규칙

| 상황 | 주어 | 예시 |
|------|------|------|
| lumi가 하는 일 | "lumi가" | "lumi가 캡션을 써드려요" |
| 사용자가 하는 일 | "사장님" / 생략 | "사진만 올리면 끝" |
| 우리 (공식) | "저희" 안 씀 | "lumi는" 또는 생략 |

## 숫자 규칙

- 시간: "37분 → 1분" (비교형)
- 가격: "월 1.9만원부터" (만원 단위)
- 횟수: "주 3개", "하루 5개" (한글)
- 비율: "2~3배" (숫자+배)
