## 2. 가치 기반 가격 분석 (Value-Based Pricing)

### 2-1. 고객이 인식하는 가치 (천장)

| 현재 대안 | 비용 | 루미 대비 |
|-----------|------|----------|
| SNS 대행사 | 월 50~100만원 | 루미는 **13~53배 저렴** |
| SNS 바이럴 마케팅 (숨고 평균) | 건당 32만원 | 월 3건이면 96만원 |
| 직접 작성 (사장님 시간) | 하루 30분 × 30일 = 15시간/월 | 시급 1만원 기준 15만원 가치 |
| ChatGPT 직접 복붙 | 무료~$20 | 편의성 차이 존재 |

**분석**: 대행사 대비 압도적 가격 우위. 그러나 직접 작성(무료) 또는 ChatGPT($20) 대비로는 "편의성 프리미엄"을 정당화해야 함.

### 2-2. 경쟁사 가격 (바닥)

| 경쟁사 | 가격 | 특징 |
|--------|------|------|
| 소셜비즈 | 월 5,000원~ | DM 자동화 특화, 캡션 생성 없음 |
| Mirra | $9~/월 (≈12,000원~) | AI 캡션, 해외 서비스 |
| Later | 무료~$5/채널 | 예약 게시, 한국어 미지원 |
| Buffer | 무료~$5/채널 | SNS 관리, 한국어 미지원 |

**분석**: 단순 예약/DM 도구는 5,000~12,000원대. 루미는 "AI 캡션 생성 + 해시태그 + 바로 게시"를 통합한 유일한 한국어 서비스로 프리미엄 정당화 가능.

### 2-3. 루미의 적정 가격 영역

```
[비용 to serve] ──── [경쟁사 5K~12K] ──── [★ 루미 19K~39K] ──── [대행사 50만~100만]
                     바닥                   현재 위치              천장
```

현재 가격은 경쟁사와 대행사 사이 **합리적 영역**에 위치. 다만 바닥(경쟁사)에 가까워 **상향 여지** 존재.

## 3. 가치 지표 (Value Metric) 분석

### 현재 가치 지표: 기능 티어 (Feature Gating)

| 평가 기준 | 현재 상태 | 판정 |
|-----------|----------|------|
| 가치와 정렬되는가? | 게시 횟수와 기능으로 분리 | 부분 정렬 |
| 이해하기 쉬운가? | 3개 플랜, 명확한 차이 | 좋음 |
| 성장에 따라 스케일하는가? | 고정 한도 (8/16/20회) | 제한적 |
| 게이밍하기 어려운가? | 해당 없음 | 좋음 |

### 핵심 가치 지표 후보

소상공인에게 가장 직관적인 가치 지표는 **"게시 횟수"**:
- "이번 달 인스타에 몇 번 올렸나" = 사장님이 체감하는 가치
- 게시 횟수가 늘면 매출 기회도 늘어남 → 가치 정렬

현재도 월 게시 한도를 두고 있지만, 플랜 간 차이가 기능 위주로 설명되어 있어 가치 지표가 흐릿함.

## 4. 티어 구조 평가 (Good-Better-Best)

### 현재 구조

| 역할 | 플랜 | 가격 | 이전 대비 가격 증가 |
|------|------|------|-------------------|
| Good (진입) | 베이직 | ₩19,000 | - |
| Better (추천) | 스탠다드 | ₩29,000 | +₩10,000 (+53%) |
| Best (프리미엄) | 프로 | ₩39,000 | +₩10,000 (+34%) |

### 문제점

1. **진입 장벽이 높다**: 베이직 ₩19,000은 소상공인에게 "일단 써볼까" 하기엔 부담. 소셜비즈(5,000원)나 무료 도구 대비 4배.
2. **가격 간격이 균일하다 (각 1만원)**: 앵커링/디코이 효과가 약함. 스탠다드가 "가성비 최고"로 느껴지지 않음.
3. **프로 플랜의 가치 점프가 약하다**: 캘린더, 트렌드, 최적 시간이 추가 1만원의 가치가 있는지 불명확.
4. **자동 갱신 없음**: 매월 재결제 → 이탈 마찰 발생. 구독 비즈니스의 핵심인 반복 매출이 불안정.

## 5. 가격 전략 제안

### 제안 A: 프리미엄 모델 유지 + 진입 장벽 완화

| 플랜 | 현재 | 제안 | 변경 이유 |
|------|------|------|----------|
| 트라이얼 | 7일/주3회 | **14일/주5회** | 충분히 써보고 결정할 시간 |
| 베이직 | ₩19,000 | **₩9,900** | 만원 미만 = 심리적 진입 장벽 대폭 낮춤 |
| 스탠다드 | ₩29,000 | **₩29,000** (유지) | 앵커 역할. 베이직 대비 3배 가치 체감 |
| 프로 | ₩39,000 | **₩49,000** | 앵커링 강화 + 프리미엄 가치 반영 |

**효과**:
- 베이직 ₩9,900 → "커피 두 잔 값" 포지셔닝, 전환율 상승 기대
- 스탠다드가 디코이 효과로 "가성비 최고"로 부각
- 프로 ₩49,000 → 앵커링으로 스탠다드를 더 합리적으로 느끼게 함
- 가격 비율: 1 : 2.9 : 4.9 (현재 1 : 1.5 : 2.1 → 차별화 강화)

### 제안 B: 연간 결제 도입

| 결제 방식 | 베이직 | 스탠다드 | 프로 |
|-----------|--------|---------|------|
| 월 결제 | ₩9,900 | ₩29,000 | ₩49,000 |
| 연간 결제 (월 환산) | ₩7,900 (20% 할인) | ₩23,000 (21% 할인) | ₩39,000 (20% 할인) |
| 연간 결제 (일시불) | ₩94,800 | ₩276,000 | ₩468,000 |

**효과**: 선불 매출 확보 + 이탈 방지 + LTV 증가

### 제안 C: 자동 갱신 전환

현재 31일 단건 결제 → **자동 갱신 구독**으로 전환 (만료 전 알림 + 원클릭 해지)

**이유**: 매월 재결제하는 모델은 이탈 마찰이 높고, 구독 비즈니스의 핵심인 MRR(월간반복매출) 예측이 불가능.

### 현재 미적용 요소

| 원칙 | 현재 상태 | 개선안 |
|------|----------|--------|

| **디코이 효과** | 스탠다드가 뚜렷한 "최적" 아님 | 베이직 ₩9,900 / 스탠다드 ₩29,000 → 3배 차이에 기능 대폭 증가 |

| **추천 표시** | 미확인 | 스탠다드에 "가장 인기" 뱃지 필수 |
| **손실 회피** | 미적용 | "이 기능 없이 게시하면 도달률 40% 낮아요" |

## 9. 결론

### 현재 가격의 강점
- 대행사 대비 압도적 가격 경쟁력 (13~53배 저렴)
- 한국어 AI 캡션 통합 서비스 = 유일한 포지셔닝
- 3-티어 Good-Better-Best 구조 기본 갖춤

### 개선 필요 사항
1. **진입 장벽 완화**: 베이직 ₩19,000은 "일단 써보자"에 비쌈 → ₩9,900 제안
2. **앵커링 강화**: 프로 ₩49,000으로 올려 스탠다드를 "베스트 딜"로 부각
3. **자동 갱신**: 단건 결제 → 구독 전환으로 MRR 안정화
4. **데이터 수집 우선**: 베타 테스터 대상 Van Westendorp 조사 후 확정

### 최우선 액션
> **베타 기간에 가격을 바꾸지 말고, 테스터 20명에게 가격 민감도를 물어라.**
> 데이터 없이 가격을 바꾸면 두 번 바꿔야 한다.
