## 미반영 항목
- **랜딩 페이지 가치 제안 명확화**: 이미 반영됨 (hero-desc에 "사진만 올리면 캡션 완성 + 해시태그 + 인스타 자동 게시" 메시지 존재)
- **베타 신청 폼 정상 동작 + 알림톡**: 이미 구현됨 (submitForm → /api/beta-apply → 알림톡 발송)
- **선착순 잔여석 실시간 표시**: 이미 구현됨 (fetchCount 함수로 /api/beta-apply GET → slot-bar 렌더링)
- **신청 완료 화면/메시지**: 이미 구현됨 (form-done 섹션에 "24시간 내 카카오톡으로 안내" 표시)
- **모바일 최적화**: 이미 구현됨 (clamp(), auto-fit 그리드, 768px/640px 미디어 쿼리)
- **20명 마감 시 대기 명단 자동 전환**: 이미 구현됨 (fetchCount에서 used >= 20 시 대기 명단 모드)
- **마감 임박 알림 ("남은 자리 5명/3명/마지막 1명")**: 단계별 임박 알림은 서버사이드 로직 필요 — 현재 잔여석 숫자 표시로 대체됨
- **제품 준비 항목 (Meta 심사, Solapi 검수 등)**: 외부 프로세스로 코드 반영 불가
- **마케팅 자산 (스크린샷, GIF, 업종별 캡션 샘플)**: 콘텐츠 제작 영역으로 코드 반영 불가
- **채널 준비 (인스타 계정, 카카오톡 채널 등)**: 외부 플랫폼 작업으로 코드 반영 불가
- **온보딩 & 피드백 (시나리오 문서화, 스프레드시트 등)**: 운영 프로세스로 코드 반영 불가
- **분석 & 추적 (전환율 추적, NPS 측정 등)**: 별도 애널리틱스 설정 필요
