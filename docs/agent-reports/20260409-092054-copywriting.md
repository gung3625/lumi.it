## 변경사항

### 4-9. Meta / SEO
- `<title>` 태그: "인스타 자동 포스팅 — 사진 한 장이면 캡션부터 게시까지 | lumi" → "lumi — 사진 한 장이면 인스타 캡션·게시 자동"
- `<meta name="description">`: "소상공인 인스타그램 자동 포스팅 서비스..." → "매장 사진 올리면 AI가 캡션·해시태그 써서 인스타에 바로 올려줘요. 월 1.9만원, 지금 무료 체험 중."
- `og:description`: "카페·음식점·뷰티샵 사장님..." → "매장 사진 올리면 캡션·해시태그·게시까지 자동. 대행사 비용의 1/25. 선착순 무료 체험 중."

### 4-1. Metrics
- "사진 1장 / 준비물 전부" → "사진 1장 / 캡션 완성까지 58초"
- "4.9점 / 테스터 평균 만족도" → "4.9/5.0 / 테스터 만족도"
- "3가지 / 매번 고를 수 있는 캡션" → "3개 / 매번 캡션 중 선택"

### 4-2. How It Works
- 섹션 타이틀: "사진 한 장이면 이 모든 게 자동이에요" → "3단계면 오늘 인스타 끝이에요"
- STEP 01 h3: "사진 올리기" → "매장 사진 올리기"
- STEP 02 h3: "lumi 사진 분석" → "lumi가 사진 읽는 중"
- STEP 03 h3: "캡션 자동 작성" → "캡션 3개 완성"
- STEP 04 h3: "인스타 자동 게시" → "인스타에 바로 업로드"

### 4-3. Features
- 섹션 라벨: "자동화가 아니라 이해" → "다른 AI랑 뭐가 다르냐면"
- 섹션 타이틀: "우리 매장 상황을 읽고 알아서 써드려요" → "동네 날씨, 근처 축제, 사장님 말투까지 반영해요"
- "글 고민 0분" 설명: 리포트 개선안 반영 (촬영 집중 메시지)
- "도달률 높이는 태그" → "노출 높이는 태그" + 설명 변경 (2~3배 노출 차이)
- "오늘 날씨가 곧 콘텐츠" 설명: 구체적 예시로 변경 ("이런 날엔 따뜻한 빵이죠")
- "최적 시간에 자동 게시" 설명: Pain 먼저 제시 ("밤 11시에 올릴 필요 없어요")
- "우리 매장 말투로만" 설명: 피드백 학습 강조로 변경
- "여러 장도 한 번에" 설명: 캐러셀 게시물 언급으로 변경

### 4-4. Demo
- 섹션 라벨: "직접 체험해 보세요" → "회원가입 없이 바로 체험"
- 섹션 타이틀: "사진 한 장 올려보세요, lumi가 바로 캡션을 써드릴게요" → "지금 매장 사진 한 장 올려보세요"
- CTA 버튼: "캡션 만들어 보기" → "내 캡션 받아보기"
- 업그레이드 라벨: "지금 이 캡션은 사진만 보고 만들었어요" → "방금 본 캡션, 이게 끝이 아니에요"
- 업그레이드 CTA: "무료 테스터로 시작하기" → "더 정교한 캡션 받아보기"

### 4-5. Proof
- 섹션 라벨: "같은 사진, 다른 글" → "같은 사진인데 반응이 다릅니다"
- 섹션 타이틀: "사장님이 쓴 글 vs lumi가 쓴 글" → "30초 만에 이렇게 달라져요"
- Before 라벨: "Before — 직접 쓴 글" → "사장님이 5분 고민해서 쓴 글"
- After 라벨: "After — lumi가 쓴 글" → "lumi가 30초 만에 쓴 글"

### 4-6. Pricing
- 섹션 타이틀: "우리 매장에 맞는 플랜을 선택하세요" → "대행사 월 50만원 vs lumi 월 1.9만원"
- 베이직 서브카피: "커피 두 잔 값으로 인스타 시작" → "일주일에 2~3번 올리는 매장"
- 스탠다드 서브카피: "사장님 10명 중 7명이 선택" → "매일 올리고 싶은 매장"
- 프로 서브카피: "대행사 없이 풀 자동화" → "인스타를 매출 채널로 키우는 매장"

### 4-7. Final CTA
- H2: "사장님, 오늘부터 인스타 걱정 끝내세요." → "내일 올릴 인스타, 오늘 1분이면 끝나요."
- 서브텍스트: "N/20 무료 테스터 모집 중" → "선착순 20명 중 N자리 남음 · 베타 기간 완전 무료 · 카드 등록 없음"
- CTA 버튼: "1분 만에 캡션 받아보기" → "남은 자리 확인하고 시작하기"
- JS 잔여석 로직: 신청자 수 표시 → 남은 자리 수(20-count) 표시로 변경
