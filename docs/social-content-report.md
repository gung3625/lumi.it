# 루미(lumi) 인스타그램 마케팅 콘텐츠 전략

## 1. 전략 개요

### 목표
- 베타 테스터 20명 확보 (1차 목표)
- 소상공인 사장님 인지도 확보 + 신뢰 구축
- "사진만 올리면 끝" 메시지 반복 각인

### 타겟 오디언스
- 한국 소상공인 (카페, 음식점, 뷰티샵, 꽃집, 베이커리, 헬스장 등)
- 인스타그램이 중요한 건 아는데 매일 올리기 귀찮은 사장님
- 대행사(월 50~100만원)는 부담스럽고, ChatGPT 복붙은 번거로운 분

### 브랜드 보이스
- 친근하고 실용적, 전문 용어 최소화
- 사장님이 바로 이해할 수 있는 쉬운 말
- 금지: 영어 남발, 기술 용어(API, SaaS), 과장 표현

---

## 2. 콘텐츠 필러 (Content Pillars)

| 필러 | 비율 | 주요 주제 |
|------|------|----------|
| 공감 콘텐츠 | 30% | 소상공인 인스타 운영 고충, "이거 나야" 공감 |
| 교육/팁 | 25% | 인스타그램 마케팅 꿀팁, 해시태그 전략, 올리기 좋은 시간 |
| 비포/애프터 | 20% | 루미 사용 전후 비교, 실제 결과물 |
| 사장님 스토리 | 15% | 업종별 성공 사례, 사장님 인터뷰 (테스터 확보 후) |
| 프로모션 | 10% | 베타 모집, 기능 소개, 가격 안내 |

---

## 3. 게시 전략

### 빈도
- 피드 게시물: 주 4~5회
- 스토리: 매일 2~3개
- 릴스: 주 2~3회

### 최적 게시 시간 (소상공인 타겟)
- 오전 7:00~8:00 (출근 전 스크롤)
- 점심 12:00~13:00 (쉬는 시간)
- 저녁 21:00~22:00 (영업 마감 후 → 핵심 시간대)

### 해시태그 전략
- 총 20~25개 / 대형 5개 + 중형 10개 + 소형 10개
- 고정 태그: #루미 #lumi #AI캡션 #인스타자동화 #소상공인마케팅
- 업종별 태그 세트 준비 (카페, 음식점, 뷰티 등)
- 트렌드 태그 주 1회 업데이트

---

## 4. 주간 콘텐츠 캘린더

| 요일 | 콘텐츠 유형 | 형식 |
|------|------------|------|
| 월 | 공감 콘텐츠 (한 주 시작 동기부여) | 카드뉴스 / 텍스트 이미지 |
| 화 | 교육/팁 (인스타 마케팅 꿀팁) | 캐러셀 (슬라이드) |
| 수 | 비포/애프터 (루미 결과물 쇼케이스) | 릴스 / 비교 이미지 |
| 목 | 사장님 스토리 or 업종별 콘텐츠 | 캐러셀 / 스토리 |
| 금 | 프로모션 or 주말 준비 팁 | 피드 이미지 + CTA |

### 게시물 4: 릴스 (짧은 영상 콘셉트)

**형식**: 15~30초 릴스
**콘셉트**: "사장님의 하루" — 바쁜 일상 중 인스타 올리는 장면
**BGM**: 트렌디한 경쾌한 음악

**스토리보드**:
- 0~5초: 사장님이 바쁘게 일하는 모습 (자막: "오늘도 정신없는 하루")
- 5~10초: 핸드폰 들고 인스타 열었다가 한숨 (자막: "인스타 또 못 올렸다")
- 10~15초: 루미 화면에 사진 올리는 모습 (자막: "사진 하나 올렸을 뿐인데")
- 15~20초: 캡션 자동 생성되는 화면 (자막: "AI가 알아서 캡션 완성")
- 20~25초: 인스타에 게시 완료 (자막: "10초 만에 인스타 끝")
- 25~30초: 루미 로고 + "무료 베타 신청 중" CTA

**캡션**:
```
영업하랴 인스타 올리랴… 사장님 하루는 48시간이어도 모자라죠.

루미 쓰면 10초면 끝나요.

프로필 링크에서 신청하세요.
```
