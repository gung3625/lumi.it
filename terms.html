이용약관

시행일 2026.01.01 · 최종 수정 2026.03.01

01

목적

이 약관은 루미(이하 "회사")가 운영하는 lumi 서비스의 이용에 관하여 회사와 이용자 간의 권리, 의무, 책임사항을 규정합니다.

02

용어 정의

  • 서비스 — 소상공인 SNS 홍보 자동화 플랫폼 lumi
  • 이용자 — 약관에 동의하고 서비스를 이용하는 자
  • 콘텐츠 — 이용자가 업로드하는 사진·텍스트 일체
  • 루미 생성물 — lumi가 자동으로 생성한 캡션·해시태그·추천
03

서비스의 내용

  • 사진 분석 기반 캡션 · 해시태그 자동 생성 (1회 업로드당 재생성 최대 3회)
  • 날씨 · 트렌드 · 업종별 맞춤 콘텐츠 반영
  • 인스타그램 피드 · 스토리 · 캐러셀 예약/승인 후 게시
  • 최적 시간 분석 · 예약 게시 (프로)
  • 말투 평가 기반 개인화 (스탠다드 이상)
  • 7일 콘텐츠 캘린더 · 트렌드 분석 (프로)
  • 월 캡션 생성 한도: 스탠다드 12회 · 프로 20회 (초과 시 다음 결제 주기까지 대기)
  • 비즈니스 플랜(DM 자동응답)은 현재 사전예약 단계이며, 정식 출시 시 별도 안내합니다.
04

루미 생성 콘텐츠 책임

lumi가 생성한 캡션·해시태그·추천은 참고 목적으로 제공되며, 정확성·적법성을 보장하지 않습니다.

최종 검토 및 게시 결정은 이용자의 책임입니다. 게시된 콘텐츠로 인한 법적 분쟁에 대해 회사는 책임을 지지 않습니다.

05

Instagram 연동

서비스는 Meta(Instagram) Graph API를 통해 동작합니다.

  • 이용자는 Meta 플랫폼 약관을 준수해야 합니다
  • Meta API 정책 변경 시 서비스 일부가 제한될 수 있습니다
  • 연동 해제는 대시보드에서 언제든 가능합니다
06

사진 데이터 처리

  • 업로드된 사진은 루미 분석 목적으로만 임시 처리됩니다
  • 제3자에게 제공하지 않습니다
  • 사진의 저작권은 이용자에게 있습니다
  • 탈퇴 시 모든 데이터는 즉시 파기됩니다
07

이용 계약

  • 약관 동의 + 회원가입 완료 시 계약 성립
  • 허위 정보, 타인 정보 도용 시 가입 거절 또는 해지
  • 서비스 운영 방해 목적 시 이용 제한
08

요금 및 결제

  • 7일 무료 체험 제공, 이후 유료 구독 자동 전환
  • 스탠다드 월 19,900원 · 프로 월 29,900원 (VAT 포함)
  • 구독 시작일 기준 매월 동일 일자 자동 결제 (토스페이먼츠)
  • 구독 취소 시 위약금 없음, 다음 결제일부터 결제 중단
  • 현재 결제 기간 만료까지 정상 이용 가능
  • 자동 결제 실패 시 이메일·알림톡으로 3회까지 재시도 안내, 최종 실패 시 유료 기능 잠김
09

청약철회 및 환불 정책 (전자상거래법 §17)

본 서비스는 전자상거래법 시행령 §21(5)에 따른 디지털콘텐츠로, 이용이 개시된 경우 청약철회가 제한됩니다.

  • 무료 체험 기간 중에는 비용이 발생하지 않습니다.
  • 유료 결제 후 월 캡션 생성 이용 0회이고 결제일로부터 7일 이내 청구 시 전액 환불합니다.
  • 캡션 생성을 1회라도 이용했거나 7일이 경과한 경우, 해당 결제 주기분은 환불되지 않습니다(디지털콘텐츠 제공 개시 간주).
  • 회사 귀책의 서비스 장애(24시간 이상)가 발생한 경우 장애 시간에 비례하여 환불합니다.
  • 미성년자가 법정대리인의 동의 없이 결제한 경우 취소·환불이 가능합니다.

11

AI 생성 콘텐츠 라이선스 및 이용 제한

  • 루미가 제공하는 캡션·해시태그·응답 초안은 참고용이며, 게시 전 회원의 검토·수정·승인이 필요합니다.
  • 회원이 생성 콘텐츠를 자신의 채널에 게시함으로써 발생하는 모든 책임(저작권, 초상권, 명예훼손, 허위·과장광고, 표시광고법 위반 등)은 회원에게 있습니다.
  • 회사는 생성 콘텐츠의 정확성·적절성·합법성을 보장하지 않습니다.
  • 저작권 귀속: AI 생성 콘텐츠의 저작자 표시·사용 권한은 회원에게 귀속됩니다. 회원은 회사에 대해, 서비스 운영·품질 개선·통계·안전성 검증·장애 분석 목적 범위 내에서 해당 콘텐츠를 저장·복제·분석·파기하는 비독점적이고 무상의 이용권을 허여합니다. 회사는 회원 식별이 가능한 형태로 외부에 재판매하지 않습니다.
  • 다음 목적의 서비스 이용을 금지합니다:
    • 특정 개인·기업·브랜드에 대한 비방 또는 허위 사실 유포
    • 욕설·혐오·차별·성적 표현, 청소년 유해 콘텐츠 생성
    • 허위광고·과대광고·사기성 콘텐츠 생성 (표시광고법 §3 위반)
    • 타인 저작물의 무단 복제·도용
  • 위반 시 사전 통지 없이 서비스 이용이 제한될 수 있으며, 회원은 위반으로 인해 제3자에게 발생한 손해에 대해 회사를 면책합니다.
12

책임 제한

회사는 다음의 사유로 발생한 손해에 대하여 책임을 지지 않습니다. 다만, 회사의 고의 또는 중대한 과실로 인한 경우에는 그러하지 아니합니다.

  • Meta(Instagram) Graph API 오류·정책 변경으로 인한 게시 실패·지연 또는 기능 제한
  • 회원 계정 정지·제한으로 인한 게시·응답 불가
  • 천재지변, DDoS, 인터넷/전력/클라우드 장애 등 불가항력
  • 회원이 AI 생성 콘텐츠를 충분한 검토 없이 게시함으로 인해 발생한 분쟁
  • 회원의 귀책(비밀번호 관리 소홀, OTP 유출 등)으로 인한 계정 피해

회사의 손해배상 책임이 인정되는 경우에도, 특약이 없는 한 책임의 범위는 회원이 직전 12개월간 회사에 지급한 구독료의 합계를 한도로 합니다 (약관법 §6·§7 범위 내).

서비스 중단 및 해지

  • 시스템 점검·유지보수 시 사전 공지 후 일시 중단 가능
  • 회사 사유로 서비스를 종료·일부 폐지하는 경우 30일 전 회원에게 공지
  • 회사가 회원의 이용 계약을 일방적으로 해지하는 경우 위반 사실과 사유를 명시해 14일 이상의 사전 이의제기 기간을 부여한 후 해지합니다(약관법 §9 참조). 다만, 법령 위반·타인 권리 침해 등 명백한 사유가 있는 경우 즉시 해지할 수 있습니다.
  • 회원은 언제든지 [설정] > [회원 탈퇴]에서 직접 해지할 수 있습니다.
