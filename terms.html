01

목적

이 약관은 루미(이하 "회사")가 운영하는 lumi 서비스의 이용에 관하여 회사와 이용자 간의 권리, 의무, 책임사항을 규정합니다.

02

용어 정의

  • 서비스 — 소상공인 SNS 홍보 자동화 플랫폼 lumi
  • 이용자 — 약관에 동의하고 서비스를 이용하는 자
  • 콘텐츠 — 이용자가 업로드하는 사진·텍스트 일체
  • AI 생성물 — lumi가 자동으로 생성한 캡션·해시태그·추천
03

서비스의 내용

  • 사진 분석 기반 캡션 · 해시태그 자동 생성
  • 날씨 · 트렌드 · 업종별 맞춤 콘텐츠 반영
  • 인스타그램 피드 · 스토리 · 캐러셀 자동 게시
  • 최적 시간 분석 · 예약 게시
  • 말투 학습 · 캡션 개인화 (스탠다드 이상)
  • 7일 콘텐츠 캘린더 · 트렌드 분석 (프로)
04

AI 생성 콘텐츠 책임

lumi가 생성한 캡션·해시태그·추천은 참고 목적으로 제공되며, 정확성·적법성을 보장하지 않습니다.

최종 검토 및 게시 결정은 이용자의 책임입니다. 게시된 콘텐츠로 인한 법적 분쟁에 대해 회사는 책임을 지지 않습니다.

05

Instagram 연동

서비스는 Meta(Instagram) Graph API를 통해 동작합니다.

  • 이용자는 Meta 플랫폼 약관을 준수해야 합니다
  • Meta API 정책 변경 시 서비스 일부가 제한될 수 있습니다
  • 연동 해제는 대시보드에서 언제든 가능합니다
06

사진 데이터 처리

  • 업로드된 사진은 AI 분석 목적으로만 임시 처리됩니다
  • 제3자에게 제공하지 않습니다
  • 사진의 저작권은 이용자에게 있습니다
  • 탈퇴 시 모든 데이터는 즉시 파기됩니다
07

이용 계약

  • 약관 동의 + 회원가입 완료 시 계약 성립
  • 허위 정보, 타인 정보 도용 시 가입 거절 또는 해지
  • 서비스 운영 방해 목적 시 이용 제한
08

요금 및 결제

  • 구독 취소 시 위약금 없음, 다음 결제일부터 중단
  • 현재 결제 기간 만료까지 정상 이용 가능

11

AI 생성 콘텐츠 이용 제한

  • 루미가 제공하는 AI 캡션은 참고용 초안이며, 게시 전 사용자의 검토 및 확인이 필요합니다.
  • 사용자가 AI 생성 캡션을 게시함으로써 발생하는 모든 책임(저작권, 명예훼손, 허위광고 등 포함)은 사용자에게 있습니다.
  • 루미는 AI 생성 콘텐츠의 정확성, 적절성, 합법성을 보장하지 않습니다.
  • 다음 목적으로 서비스를 사용하는 것을 금지합니다:
    • 특정 개인, 기업, 브랜드에 대한 비방 또는 허위 정보 생성
    • 욕설, 혐오 표현, 차별적 콘텐츠 생성
    • 허위 광고, 과대 광고, 사기성 콘텐츠 생성
  • 위반 시 사전 통지 없이 서비스 이용이 제한될 수 있습니다.
12

자동 게시 면책

인스타그램 자동 · 예약 게시는 Meta API에 의존하며, 다음 상황에서 회사는 책임을 지지 않습니다.

  • API 오류로 인한 게시 실패 · 지연
  • Meta 정책 변경으로 인한 기능 제한
  • 이용자 계정 정지로 인한 게시 불가
