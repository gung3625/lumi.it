1

Facebook 개인 계정

1분 · PC 또는 모바일

이미 페이스북을 쓰신 적 있으면 건너뛰셔도 OK. 본인 명의 계정 1개만 있으면 됩니다.

아직 계정이 없어요
  1. facebook.com/r.php 접속
  2. 이름·생일·이메일(또는 휴대폰)·비밀번호 입력 → "가입하기"
  3. 이메일 또는 SMS 인증 코드 입력
  4. 완료!
💡 매장명이 아닌 본인 이름으로 가입하세요. 페이스북 페이지(다음 단계)는 별도로 매장명으로 만들어요.
본인 계정인지 확인하는 방법
  1. facebook.com 접속 → 로그인
  2. 오른쪽 상단의 본인 프로필 사진 클릭 → 본인 이름이 보이면 OK
다음: Facebook 페이지 만들기 →
2

매장 이름의 Facebook 페이지 만들기

2-3분 · PC 권장 (모바일도 가능)

사장님 매장 이름으로 페이지를 하나 새로 만들어요. 인스타 비즈니스 계정은 이 페이지와 1:1로 연결되어야 동작합니다. 매장 영업과 무관 — 그냥 연결용이라고 생각하시면 돼요.

💻 PC 에서 만들기 — 가장 안정적, 권장

준비물: 개인 페이스북 계정 로그인 상태

  1. 시작 페이지로 이동
    facebook.com/pages/create 접속
    또는: 왼쪽 사이드바 "페이지" → "새 페이지 만들기"
  2. 기본 정보 입력 (왼쪽 폼)
    • 페이지 이름: 매장명 그대로. LLC, 주식회사 등은 빼도 됩니다
    • 카테고리: 최대 3개 선택. "카페" 입력하면 자동완성 → 가장 가까운 것
    • 설명: 255자 이내, 비워도 OK (나중에 수정 가능)
    오른쪽 미리보기에 이름·카테고리 실시간 반영됨
  3. "페이지 만들기" 클릭 (왼쪽 하단 파란 버튼)
  4. 프로필 사진·커버 추가 (선택, 건너뛰기 OK)
    • 프로필: 170×170px 권장, 원형으로 잘림 → 로고 중앙 배치
    • 커버: 1640×856px 권장, 매장 외관·대표 메뉴
  5. 사용자 이름 (@username) 설정
    생성 직후 뜨는 "사용자 이름 만들기" → facebook.com/내매장명 형태
    중복 시 초록 체크 X. 나중에도 설정에서 변경 가능
  6. 기본 정보 채우기 (선택)
    관리 페이지 → 왼쪽 "정보" → 전화·웹사이트·영업시간
    이 정보가 구글 검색에도 노출됨
  7. 행동 유도 버튼 (CTA) 추가 (선택)
    페이지 상단 "버튼 추가" → "메시지 보내기" / "전화하기" / "웹사이트 방문" 중 선택
  8. 페이지 공개 확인
    새 페이지는 바로 공개 상태. "설정 → 공개 범위" 에서 비공개 전환 가능
💡 이름에 동네명을 붙이면 검색에 유리 — "홍대 수민이네", "강남 카페모카" 처럼.
💡 카테고리는 나중에도 바꿀 수 있으니 처음엔 가장 가까운 거 하나 골라도 OK.
📱 모바일 (iOS / Android 앱) — 흐름 같음, 버튼 위치만 조금 다름
  1. 앱 실행 → 메뉴
    • iPhone: 오른쪽 하단 ≡
    • Android: 오른쪽 상단 ≡
  2. 메뉴에서 "페이지" 선택
  3. "+ 만들기" 또는 "새 페이지" 탭
  4. 시작하기 → 이름 입력 → "다음"
  5. 카테고리 선택 (최대 3개, 검색으로 찾기) → "다음"
  6. 소개 작성 (선택, 255자) → "다음" 또는 "건너뛰기"
  7. "페이지 만들기" 탭 → 즉시 생성
  8. 이어지는 온보딩 (전부 건너뛰기 OK)
    • 프로필 사진 추가
    • 커버 사진 추가
    • WhatsApp 연결
    • 친구 초대
  9. 마무리 설정: 페이지 상단 "프로필 편집" → 사용자 이름 / CTA 버튼 추가
    "설정 → 페이지 정보" 에서 주소·영업시간 입력
💡 사진은 나중에 고화질로 교체하세요. 처음엔 스킵하고 이름·카테고리만 넣어도 인스타 연동은 가능합니다.
자주 묻는 질문
Q. 페이스북 페이지를 누가 봐요? 친구들이 봐요?
아무도 안 봐도 됩니다. 페이지는 인스타 연결용이지, 사장님이 적극적으로 운영할 필요 없어요. 비공개 페이지로 두셔도 인스타 연동은 작동해요.
Q. 페이지가 여러 개 있으면?
OK — 매장 이름과 일치하는 페이지를 인스타와 연결하시면 됩니다.
Q. 매장이 여러 곳이면?
각 매장마다 별도 페이지 + 별도 인스타 계정 → 별도 lumi 계정으로 가입하셔야 해요.

3

인스타를 비즈니스 계정으로 전환

2-3분 · 모바일 인스타 앱에서

매장 운영용 인스타를 비즈니스 또는 크리에이터 계정으로 전환해요. 개인 계정으로는 lumi 가 게시할 수 없어요 (Meta 정책).

⚠️ 전환해도 친구·팔로워는 그대로 봐요. 인스타 노출도 똑같음. 단지 인사이트·광고·외부 도구 (lumi 같은) 가 추가로 작동.
📱 iOS / Android 앱 — 가장 안정적, 권장
  1. 인스타그램 앱 열기
  2. 하단 탭바의 본인 프로필 탭
  3. 두 가지 경로 중 하나 선택:
    • 빠른 길: 프로필 화면의 "프로필 편집" 누르고 아래로 스크롤 → "프로페셔널 계정으로 전환" 보임
    • 정석 경로: 우측 상단 ≡ (햄버거 메뉴) → "설정 및 활동" → "계정 유형 및 도구" (또는 "비즈니스 도구 및 관리") → "프로페셔널 계정으로 전환"
  4. 안내 화면 → "계속"
  5. 카테고리 선택 — 매장 종류와 가까운 것 (예: "카페", "음식점", "소매업"). 검색창에 키워드 입력으로 빠르게 찾기.
  6. 비즈니스 ✓ 선택 → "다음" (크리에이터도 가능하나 쇼핑·광고 활용에 비즈니스가 유리)
  7. 연락처 정보 확인 (전화·이메일·주소) → 공개/비공개 선택 가능 → "다음"
  8. (마지막 화면이 Facebook 페이지 연결) → Step 4 로 이어집니다 ↓
💡 메뉴 이름이 인스타 버전마다 달라요 ("계정 유형 및 도구" / "비즈니스 도구 및 관리" 등). "프로페셔널" 또는 "비즈니스" 키워드로 검색하면 가장 빠릅니다.
💻 PC (Chrome / Edge 등 데스크톱 브라우저) — 큰 화면에서 빠르게
  1. instagram.com 로그인 (데스크톱 브라우저 권장 — 모바일 웹은 메뉴 숨겨짐)
  2. 왼쪽 사이드바 맨 아래 ≡ 더 보기 (More) → "설정 (Settings)"
  3. 설정 화면 왼쪽 메뉴 스크롤 → "계정 유형 및 도구 (Account type and tools)"
  4. "프로페셔널 계정으로 전환 (Switch to professional account)" → 안내 → "계속"
  5. 비즈니스 선택 → "다음" (앱과 순서 다름: PC 는 비즈니스 먼저, 카테고리 다음)
  6. 카테고리 검색창에 "카페", "음식점" 등 입력 → 선택 → "완료"
  7. (PC 에서는 연락처 입력 화면 간소화) → 필요시 나중에 모바일 앱에서 추가
  8. Facebook 페이지 연결: PC 에서는 "나중에 하기" 기본 → 모바일 앱에서 마지막 단계 진행 권장
    또는: 설정 → 계정 센터 → 연결된 계정 → Facebook 페이지 추가
💡 PC 장단점 — ⭕ 큰 화면 카테고리 검색 편함, 1분 컷. ⚠️ 연락처 버튼 / 주소 지도 연동 / 쇼핑 태그 활성화는 모바일에서만 세밀 조정 가능. 페이지 연결은 모바일이 더 안정적.

4

인스타 ↔ Facebook 페이지 연결

1분 · 보통 Step 3 도중에 자동으로 나옴

Step 3 의 비즈니스 전환 마지막 단계에서 자동으로 "Facebook 페이지 연결" 화면이 떠요. Step 2 에서 만든 매장 페이지를 선택하면 끝.

비즈니스 전환 중에 연결 (가장 흔한 경우)
  1. Step 3 의 마지막 화면 = "Facebook 페이지 연결"
  2. 본인 페이스북에 로그인되어 있어야 페이지 리스트가 떠요 — 페이스북 로그인 X 면 먼저 로그인 진행
  3. Step 2 에서 만든 매장 페이지를 선택 → "연결" 탭
  4. 완료! 인스타 프로필에 페이지 이름이 보이면 성공.
비즈니스 전환은 마쳤는데 페이지 연결을 건너뛴 경우
  1. 인스타 앱 → 하단 본인 프로필 탭
  2. "프로필 편집" 누르고 아래로 스크롤 → "페이지" 또는 "Facebook 페이지" 항목
  3. "페이지 연결" 또는 "Facebook 페이지 선택" → 위에서 만든 매장 페이지 선택
  4. 연결 완료
💡 또는 우측 상단 ≡ → 설정 및 활동 → 계정 유형 및 도구 → "Facebook 페이지 연결" 안에서도 같은 작업 가능.
"내 페이지가 안 보여요"
  1. 먼저 페이스북 본인 계정으로 로그인되어 있는지 (브라우저 또는 페이스북 앱)
  2. Step 2 에서 만든 페이지의 관리자가 본인 계정인지 확인 — facebook.com/pages 에서 "내 페이지"에 떠야 함
  3. 페이지가 1주일 이내 새로 만들어졌다면 일시적으로 안 보일 수 있어요. 24시간 후 재시도.
이미 다른 인스타와 연결된 페이지 같다고 떠요
  1. 1개 페이지에 인스타 1개만 연결 가능 — 옛 인스타와 연결 해제 후 새 인스타로 연결
  2. 또는 다른 페이지를 새로 만들어서 연결 (Step 2 반복)
