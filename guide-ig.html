1

Facebook 개인 계정

1분 · PC 또는 모바일

이미 페이스북을 쓰신 적 있으면 건너뛰셔도 OK. 본인 명의 계정 1개만 있으면 됩니다.

아직 계정이 없어요
  1. facebook.com/r.php 접속
  2. 이름·생일·이메일(또는 휴대폰)·비밀번호 입력 → "가입하기"
  3. 이메일 또는 SMS 인증 코드 입력
  4. 완료!
💡 매장명이 아닌 본인 이름으로 가입하세요. 페이스북 페이지(다음 단계)는 별도로 매장명으로 만들어요.
본인 계정인지 확인하는 방법
  1. facebook.com 접속 → 로그인
  2. 오른쪽 상단의 본인 프로필 사진 클릭 → 본인 이름이 보이면 OK
다음: Facebook 페이지 만들기 →
2

매장 이름의 Facebook 페이지 만들기

2-3분 · PC 권장 (모바일도 가능)

사장님 매장 이름으로 페이지를 하나 새로 만들어요. 인스타 비즈니스 계정은 이 페이지와 1:1로 연결되어야 동작합니다. 매장 영업과 무관 — 그냥 연결용이라고 생각하시면 돼요.

PC 에서 만들기 (2분, 권장)
  1. facebook.com/pages/create 접속 (로그인 상태)
  2. 왼쪽 입력란:
    • 페이지 이름: 매장 이름 (예: "수민이네 카페", "강남 OO 커피")
    • 카테고리: 검색창에 매장 종류 타이핑 (예: "카페", "음식점") → 자동완성 중 선택
    • 설명: 비워두셔도 OK (나중에 채울 수 있음)
  3. 오른쪽에 미리보기 — 페이지 이름·카테고리 확인
  4. "페이지 만들기" 클릭
  5. (선택) 프로필·커버 사진 업로드 — 건너뛰기 OK
  6. 완료! 페이지 주소 (facebook.com/매장이름) 가 생겼어요.
💡 매장 이름이 너무 흔하면 동네명 추가 — "강남 수민이네", "홍대 카페모카" 처럼.
💡 카테고리는 정확하지 않아도 OK. 나중에 페이지 설정에서 변경 가능.
모바일에서 만들기 (앱)
  1. Facebook 앱 열기 → 우측 하단 ≡ 메뉴
  2. "페이지" → "만들기" (또는 "+ 새 페이지")
  3. 이름 입력 → 카테고리 선택 → "만들기"
  4. 완료!
자주 묻는 질문
Q. 페이스북 페이지를 누가 봐요? 친구들이 봐요?
아무도 안 봐도 됩니다. 페이지는 인스타 연결용이지, 사장님이 적극적으로 운영할 필요 없어요. 비공개 페이지로 두셔도 인스타 연동은 작동해요.
Q. 페이지가 여러 개 있으면?
OK — 매장 이름과 일치하는 페이지를 인스타와 연결하시면 됩니다.
Q. 매장이 여러 곳이면?
각 매장마다 별도 페이지 + 별도 인스타 계정 → 별도 lumi 계정으로 가입하셔야 해요.
다음: 인스타 비즈니스 전환 →
3

인스타를 비즈니스 계정으로 전환

2-3분 · 모바일 인스타 앱에서

매장 운영용 인스타를 비즈니스 또는 크리에이터 계정으로 전환해요. 개인 계정으로는 lumi 가 게시할 수 없어요 (Meta 정책).

⚠️ 전환해도 친구·팔로워는 그대로 봐요. 인스타 노출도 똑같음. 단지 인사이트·광고·외부 도구 (lumi 같은) 가 추가로 작동.
iOS / Android 앱 — 같은 순서
  1. 인스타그램 앱 열기
  2. 하단 탭바의 본인 프로필을 누르세요 — 이 화면이 곧 "설정 및 활동" 입니다 (별도로 햄버거 메뉴 안 거쳐도 됨)
  3. "계정 유형" 탭 (스크롤 좀 내려야 보일 수 있음)
  4. 그 안에서 "프로페셔널 계정으로 변경" 선택
  5. 안내 화면 → "계속"
  6. 카테고리 선택 — 매장 종류와 가까운 것 (예: "카페", "음식점", "소매업"). 검색창 활용 OK.
  7. 비즈니스 ✓ 선택 → "다음" (크리에이터도 가능하나 비즈니스 권장)
  8. 연락처 정보 검토 (전화·이메일·주소) → 안 보이게 할 수 있음 → "다음"
  9. (다음 화면이 Facebook 페이지 연결) → Step 4 로 이어집니다 ↓
💡 인스타 버전에 따라 "계정 유형" 메뉴가 우측 상단 ≡ (햄버거 메뉴) 안에 있을 수도 있어요. "프로페셔널", "비즈니스 도구", "계정 전환" 등 키워드로 찾으시면 됩니다.

"계정 유형" 또는 "프로페셔널 계정으로 변경" 메뉴가 안 보여요
  1. 인스타 앱이 최신 버전인지 확인 (앱스토어/플레이스토어 업데이트)
  2. 본인 계정으로 로그인됐는지 (다중계정이면 매장 계정으로 전환)
  3. 그래도 안 보이면 — 가끔 인스타가 일부 사용자에게 일시 비활성화하는 경우 있어요. 24시간 후 재시도.

4

인스타 ↔ Facebook 페이지 연결

1분 · 보통 Step 3 도중에 자동으로 나옴

Step 3 의 비즈니스 전환 마지막 단계에서 자동으로 "Facebook 페이지 연결" 화면이 떠요. Step 2 에서 만든 매장 페이지를 선택하면 끝.

비즈니스 전환 중에 연결 (가장 흔한 경우)
  1. Step 3 의 마지막 화면 = "Facebook 페이지 연결"
  2. 본인 페이스북에 로그인되어 있어야 페이지 리스트가 떠요 — 페이스북 로그인 X 면 먼저 로그인 진행
  3. Step 2 에서 만든 매장 페이지를 선택 → "연결" 탭
  4. 완료! 인스타 프로필에 페이지 이름이 보이면 성공.
비즈니스 전환은 마쳤는데 페이지 연결을 건너뛴 경우
  1. 인스타 앱 → 하단 내 프로필 (= 설정 및 활동)
  2. "계정 유형" 또는 "Facebook 페이지 연결" 항목 탭
  3. "페이지 연결" 또는 "Facebook 페이지 선택" → 매장 페이지 선택
  4. 연결 완료
💡 또는 인스타 프로필 그리드 화면의 "프로필 편집" 버튼 안에서도 "페이지" 항목으로 같은 작업 가능.
"내 페이지가 안 보여요"
  1. 먼저 페이스북 본인 계정으로 로그인되어 있는지 (브라우저 또는 페이스북 앱)
  2. Step 2 에서 만든 페이지의 관리자가 본인 계정인지 확인 — facebook.com/pages 에서 "내 페이지"에 떠야 함
  3. 페이지가 1주일 이내 새로 만들어졌다면 일시적으로 안 보일 수 있어요. 24시간 후 재시도.
이미 다른 인스타와 연결된 페이지 같다고 떠요
  1. 1개 페이지에 인스타 1개만 연결 가능 — 옛 인스타와 연결 해제 후 새 인스타로 연결
  2. 또는 다른 페이지를 새로 만들어서 연결 (Step 2 반복)
