프로필 링크

인스타 프로필엔 링크 1개만 넣을 수 있어요. 매장의 메뉴·예약·배달·지도 등 모든 외부 링크를 이 페이지에 모아드릴게요.

ℹ️ 왜 한 번은 직접 해야 하나요?

인스타는 10년 넘게 캡션 안의 URL 을 클릭 불가능으로 막아왔어요. 캡션에 "lumi.it.kr/r/내주소" 적어도 손님이 직접 복사해야 함 → 효과 약함.

유일하게 1탭 클릭 가능한 곳 = 프로필 bio 링크. 손님이 매장 프로필 들어가서 "🔗 웹사이트" 누르면 즉시 외부 사이트로 직행.

근데 인스타 정책상 bio 링크는 API 로 수정 불가 — lumi 가 자동으로 못 바꿔요. 그래서 이 주소를 사장님이 한 번만 bio 에 붙여넣어주세요. 이후 메뉴·예약·배달 링크 추가·교체는 이 페이지에서 자유롭게 — 인스타 bio 는 안 건드려도 다 반영됩니다.

주소 준비 중…

위 주소를 복사해서 인스타 앱 → 프로필 편집 → "웹사이트" 칸에 한 번만 붙여넣으세요. (이후 평생 그대로 둬도 OK)
