인스타그램 연동

lumi가 인스타그램에 게시물을 올리려면
아래 3단계 연동이 필요해요

비용 0원 · 3분이면 완료
1

비즈니스 계정으로 전환

인스타그램 계정을 비즈니스 계정으로 바꿔야 해요.
개인 계정으로는 외부 앱에서 게시할 수 없어요.
아래 버튼을 누르면 전환 페이지로 바로 이동해요.

전환하러 가기
2

페이스북 페이지 만들기

인스타그램 비즈니스 계정은 페이스북 페이지와 연결해야 해요.
페이지 이름은 매장 이름으로 만들면 돼요.
이미 있다면 그대로 사용하면 됩니다.

만들러 가기
3

lumi에서 연동하기

마지막이에요! 아래 버튼을 누르면
인스타그램 로그인 창이 뜨고, 허용하면 연동 완료!

연동이 완료되면
lumi의 모든 기능을 사용할 수 있어요!
