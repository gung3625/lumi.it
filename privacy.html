02

사진 데이터 처리

이용자가 업로드한 사진은 AI 캡션 생성을 위해 OpenAI API로 전송됩니다.

  • 전송 형태: base64 인코딩 이미지
  • OpenAI는 API 데이터를 모델 학습에 사용하지 않습니다 (Zero Data Retention)
  • 분석 완료 후 서버에 사진을 저장하지 않습니다
  • 사진의 저작권은 이용자에게 있습니다
03

Instagram 토큰 관리

Instagram 연동 시 OAuth 인증을 통해 액세스 토큰을 발급받습니다.

  • 토큰은 암호화하여 Supabase(서버 전용) 보관, 클라이언트 미반환
  • 접근 권한: 피드/스토리 게시, 미디어 관리, 댓글 관리
  • 연동 해제 또는 탈퇴 시 토큰 즉시 삭제
  • 팔로워 목록·연락처 수집은 일체 하지 않습니다

05

날씨 · 위치 정보

날씨 반영 캡션 생성 시 기상청 API를 호출합니다.

  • 이용자의 매장 지역 정보를 격자좌표로 변환하여 조회
  • 정확한 주소나 GPS 좌표를 수집하지 않습니다
  • 기상청에 개인정보를 전달하지 않습니다
06

제3자 제공

원칙적으로 개인정보를 제3자에게 제공하지 않습니다.

  • 이용자가 사전 동의한 경우
  • 법령에 따른 수사기관 요청

위 경우를 제외하고 어떠한 목적으로도 제3자에게 제공하지 않습니다.

07

이용자(정보주체)의 권리

회원은 개인정보보호법 §4·§35·§36·§37에 따라 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요구
  • 오류 정정·삭제 요구
  • 처리 정지 요구
  • 동의 철회 · 회원 탈퇴
  • 개인정보 이동권(데이터 포터빌리티) 요청

행사 방법:

  • [설정] > [회원 탈퇴]에서 본인 확인 후 직접 즉시 처리
  • 열람·정정·이동 요청: lumi@lumi.it.kr (처리 부서: 대표 직결, 접수 후 10일 이내 회신)

대리인이 요청하는 경우 위임장과 인감증명서를 제출해야 합니다. 회사는 법정 사유가 있는 경우 요청을 제한할 수 있으며, 제한 시 사유를 지체 없이 통지합니다.

07-A

데이터 삭제 안내

루미는 사용자가 자신의 데이터를 언제든지 삭제할 수 있도록 다음 절차를 제공합니다.

1. 앱 내 자동 삭제 (권장)

설정 → "계정" 섹션 → "회원 탈퇴" 버튼을 누르면 즉시 처리됩니다.

탈퇴 즉시 영구 삭제되는 데이터:

  • 회원 정보 (이름, 이메일, 카카오 ID, 사업자 정보)
  • 연결된 SNS 계정 토큰 (Instagram · TikTok)
  • 업로드한 사진·영상·캡션 히스토리
  • 예약 게시 일정
  • 브랜드 보이스 학습 데이터
  • 트렌드 평가·캡션 평가 기록

처리 시간: 즉시 (DB 트리거). 개인정보보호법 §36에 따라 30일 이내 백업 본까지 삭제됩니다.

2. 이메일 요청

gung3625@gmail.com으로 "데이터 삭제 요청" 제목으로 발송해 주세요. 본인 확인 후 7일 이내 처리합니다.

3. Instagram 데이터 분리 삭제

루미 계정은 유지하면서 Instagram 연결만 해제하려면: 설정 → "Instagram" → "연결 해제"

  • 연결 해제 즉시 루미에 저장된 IG 토큰·인사이트 캐시 삭제
  • 루미가 IG에 게시한 콘텐츠는 Instagram 본인 계정에서 직접 관리

4. TikTok 데이터 분리 삭제

설정 → "TikTok" → "연결 해제" — 연결 해제 즉시 토큰·캐시 삭제

삭제 요청 또는 문의: gung3625@gmail.com
