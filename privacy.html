02

사진 데이터 처리

이용자가 업로드한 사진은 AI 캡션 생성을 위해 OpenAI API로 전송됩니다.

  • 전송 형태: base64 인코딩 이미지
  • OpenAI는 API 데이터를 모델 학습에 사용하지 않습니다 (Zero Data Retention)
  • 분석 완료 후 서버에 사진을 저장하지 않습니다
  • 사진의 저작권은 이용자에게 있습니다
03

Instagram 토큰 관리

Instagram 연동 시 OAuth 인증을 통해 액세스 토큰을 발급받습니다.

  • 토큰은 암호화하여 Supabase(서버 전용) 보관, 클라이언트 미반환
  • 접근 권한: 피드/스토리 게시, 미디어 관리, DM 수신(비즈니스 플랜 한정)
  • 연동 해제 또는 탈퇴 시 토큰 즉시 삭제
  • 팔로워 목록·연락처 수집은 일체 하지 않습니다
03-1

DM · 댓글 본문 처리 (AI 자동응답)

회원이 Meta 플랫폼(Instagram)에서 수신한 DM 및 댓글 본문을 AI 자동응답·트렌드 분석 목적으로 처리합니다.

  • 회원은 고객(DM 발신자)에 대해 AI 응답 고지·동의 확보의 책임을 부담합니다. Lumi는 개인정보보호법 §26에 따른 수탁 처리자입니다.
  • PII(전화번호·주민등록번호·이메일·카드번호)는 서버 진입 시점에 정규식 기반 자동 마스킹 후 AI 모델로 전달합니다.
  • 전송 구간 TLS 1.3 암호화 + 저장 구간 AES-256 암호화를 적용합니다.
  • DM/댓글 접근 로그는 90일 동안 보관한 뒤 자동 폐기됩니다.
  • AI가 생성한 응답 초안은 비즈니스 플랜(Shadow 모드)에서는 저장만 되고 실발송되지 않습니다. 정식 발송은 회원 검토 후 수동 승인 또는 명시적 자동 발송 설정 시에만 이루어집니다.
  • 본 서비스는 Meta Platform Terms · Developer Policies를 준수합니다.

05

날씨 · 위치 정보

날씨 반영 캡션 생성 시 기상청 API를 호출합니다.

  • 이용자의 매장 지역 정보를 격자좌표로 변환하여 조회
  • 정확한 주소나 GPS 좌표를 수집하지 않습니다
  • 기상청에 개인정보를 전달하지 않습니다
06

제3자 제공

원칙적으로 개인정보를 제3자에게 제공하지 않습니다.

  • 이용자가 사전 동의한 경우
  • 법령에 따른 수사기관 요청

위 경우를 제외하고 어떠한 목적으로도 제3자에게 제공하지 않습니다.

07

이용자(정보주체)의 권리

회원은 개인정보보호법 §4·§35·§36·§37에 따라 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요구
  • 오류 정정·삭제 요구
  • 처리 정지 요구
  • 동의 철회 · 회원 탈퇴
  • 개인정보 이동권(데이터 포터빌리티) 요청

행사 방법:

  • [설정] > [회원 탈퇴]에서 본인 확인 후 직접 즉시 처리
  • 열람·정정·이동 요청: lumi@lumi.it.kr (처리 부서: 대표 직결, 접수 후 10일 이내 회신)

대리인이 요청하는 경우 위임장과 인감증명서를 제출해야 합니다. 회사는 법정 사유가 있는 경우 요청을 제한할 수 있으며, 제한 시 사유를 지체 없이 통지합니다.
